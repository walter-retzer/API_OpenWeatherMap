# API_OpenWeatherMap
Aplicativo consumindo Informações meteorológicas da API CurrentWeatherData, utilizando a biblioteca Retrofit, conceitos de MVVM e Kotlin.
Tela de Abertura com animações utilizando a biblioteca Lottie.
